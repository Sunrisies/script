# Bun.js 脚本集合

这是一个基于Bun.js的脚本集合项目，旨在提供各种实用脚本，方便在其他地方使用。

## 项目特点

- 使用Bun.js作为运行时环境
- 不依赖第三方库，保持轻量
- 提供多种实用脚本工具

## 安装和使用

1. 确保已安装[Bun](https://bun.sh/)
2. 克隆此仓库
3. 根据需要运行相应的脚本

```bash
bun run <脚本名称>
```

## 脚本列表

- [文件操作](./scripts/file.js) - 提供文件和目录操作功能
- [网络请求](./scripts/network.js) - 提供HTTP请求功能
- [数据处理](./scripts/data.js) - 提供数据处理和转换功能
- [系统工具](./scripts/system.js) - 提供系统相关工具
- [时间日期](./scripts/datetime.js) - 提供时间和日期处理功能
